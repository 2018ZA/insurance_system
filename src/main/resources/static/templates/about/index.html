Описание проекта

InsurancePro - это полнофункциональная информационная система для автоматизации деятельности страховой компании. Система разработана как дипломный проект и демонстрирует навыки работы с современными технологиями разработки веб-приложений.

Основные возможности системы:
  • Управление клиентской базой (CRUD операции)
  • Оформление договоров страхования различных типов
  • Учет платежей и страховых случаев
  • Разграничение прав доступа по ролям пользователей
  • Расширенная статистика и аналитика
  • Генерация отчетов и экспорт данных
  • Адаптивный интерфейс для работы на любых устройствах
Архитектура проекта:

Проект построен по трехзвенной архитектуре (MVC) с использованием современных паттернов проектирования. Серверная часть реализована на Spring Boot, что обеспечивает высокую производительность и масштабируемость. Клиентская часть использует современные стандарты HTML5 и CSS3 с адаптивным дизайном.
